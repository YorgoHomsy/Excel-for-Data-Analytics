Key Features Learned
-------------------
AutoFit Columns
---
Automatically adjust column width to fit data by double-clicking between column headers.

Cell Formatting
-
Use the Ribbon to change cell formats from General to Number, Date, Text, and other formats.

Sort & Filter
-
Apply sorting and filtering to datasets

Use dropdown menus to show or hide specific values

Commonly used for database-style tables

Data Visualization
-
Create charts using Recommended Charts

Preview chart types before selecting one

Chart Management
-
Move charts to a separate worksheet using Chart Design → Move Chart → New Sheet

Name chart worksheets for better organization

Clear Filters
-
Remove applied filters via Data → Sort & Filter → Clear

Freeze Panes
-
Freeze rows and columns for easier navigation

Example: selecting B2 freezes Row 1 and Column A

Options include Freeze Top Row and Freeze First Column
